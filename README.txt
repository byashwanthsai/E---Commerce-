# E-Commerce System (Java CLI)

A simple Java-based Command Line E-Commerce application that lets users view products, add items to a cart with quantities, and view cart totals. Designed to demonstrate core Java programming concepts through a menu-driven interface.

## Features
- View available products and prices  
- Add products to cart with quantity selection  
- View cart with item-wise and grand total  
- Simple and interactive CLI menu  

## Concepts Used
- Java fundamentals  
- ArrayList collections  
- Loops and conditional statements  
- Scanner for user input  
- DecimalFormat for formatted output  

## How to Run
```bash
javac EcommerceSystem.java
java EcommerceSystem
